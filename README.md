# HepTech Admin Dashboard

A job tracking dashboard built for HepTech.

This internal tool allows technicians to create, manage and track jobs.

## Features

- Create job records
- Track job status
- Add notes
- Attach photos and videos
- Search jobs
- Local browser storage

## Technologies

- HTML
- CSS
- JavaScript
- LocalStorage
- IndexedDB
